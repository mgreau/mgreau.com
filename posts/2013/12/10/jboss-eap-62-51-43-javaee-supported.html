List of all Java EE JSRs supported

Supported in JBoss EAP 4.3 / 5 /6

  • JSR 052: A Standard Tag Library for JavaServer Pages

  • JSR 067: JavaTM APIs for XML Messaging 1.0

  • JSR 093: JavaTM API for XML Registries 1.0 (JAXR)

  • JSR 101: JavaTM APIs for XML based RPC

  • JSR 153: Enterprise JavaBeansTM 2.1

  • JSR 181: Web Services Metadata for the JavaTM Platform

  • JSR 220: Enterprise JavaBeansTM 3.0

  • JSR 914: JavaTM Message Service (JMS) API

  • JSR 919: JavaMail

Supported in JBoss EAP 4.3

  • JSR 151: JavaTM 2 Platform, Enterprise Edition 1.4 (J2EE 1.4) Specification

  • JSR 88: JavaTM EE Application Deployment

  • JSR 109: Implementing Enterprise Web Services

  • JSR 115: JavaTM Authorization Contract for Containers

  • JSR 154: JavaTM Servlet 2.4 Specification

  • JSR 224: JavaTM API for XML-Based Web Services (JAX-WS) 2.0

  • JSR 907: JavaTM Transaction API (JTA)

Supported in JBoss EAP 4.3 / 5.1

  • JSR 077: J2EETM Management

  • JSR 112: J2EETM Connector Architecture 1.5

  • JSR 220: Enterprise JavaBeansTM 3.0

  • JSR 222: JavaTM Architecture for XML Binding (JAXB) 2.0

  • JSR 245: JavaServerTM Pages 2.1

  • JSR 250: Common Annotations for the JavaTM Platform

  • JSR 252: JavaServer Faces 1.2

Supported in JBoss EAP 5.1

  • JSR 244: JavaTM Platform, Enterprise Edition 5 (Java EE 5) Specification

  • JSR 224: JavaTM API for XML-Based Web Services (JAX-WS) 2.0

  • JSR 311: JAX-RS: The JavaTM API for RESTful Web Services

Supported in JBoss EAP 5.1 / 6

  • JSR 907: JavaTM Transaction API (JTA)

Supported in JBoss EAP 6
